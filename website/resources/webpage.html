Guide on how to setup Boodle

In this section you will be granted an easy amd guick guide on on how to setup Boodle on your Discord server in a few simple steps.

Step 1 - Log into your Discord account

The first thing you will have to do is click on the following button, where you will have to log into your Discord account:

Step 2 - Connect Boodle to your Discord server

After you have logged in you will be presented by a page similar to this:



Choose which server you want Boodle on in the dropdown menu at "Add to server". Click continue.

Step 3 - Setup Boodle on your Discord server

When you have added the bot to the desired discord server, you will need to set it up personally, so it can start sending you the Moodle information that you desire. To setup Boodle on your Discord server you will have to write the command /setup in the chat, and click on the link that Boodle sends in the chat.

Commands for Boodle

In this section you will see all the possible commands that you can use with Boodle on Discord.

Command Description
/setup /setup starts the process of setting up the users data in the database.
/courses /courses displays the next 3 courses of the user.
/Weekday /Weekday allows the user to decide for which day they want the course material.
